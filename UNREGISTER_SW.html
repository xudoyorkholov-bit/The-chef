üîß Service Worker Muammosini Hal Qilish

Muammo: Eski Service Worker API so'rovlarini bloklayapti va "net::ERR_FAILED" xatosini keltirib chiqarayapti.

Yechim 1: Avtomatik o'chirish

Yechim 2: Qo'lda o'chirish

  1. Brauzerda F12 tugmasini bosing (Developer Tools)
  2. Console tabiga o'ting
  3. Quyidagi kodni nusxalang va Console ga yozing: 
    navigator.serviceWorker.getRegistrations().then(registrations => {
  registrations.forEach(registration => {
    registration.unregister();
    console.log('Service Worker o\'chirildi');
  });
});
  4. Enter tugmasini bosing
  5. Sahifani qayta yuklang (Ctrl+Shift+R yoki Cmd+Shift+R)

Yechim 3: Incognito Mode

Yoki shunchaki Incognito/Private rejimda oching:

  • Chrome: Ctrl+Shift+N
  • Firefox: Ctrl+Shift+P
  • Safari: Cmd+Shift+N

Yechim 4: Cache ni to'liq tozalash

  1. Brauzer sozlamalariga o'ting
  2. "Clear browsing data" yoki "Tarixni tozalash" ni toping
  3. "Cached images and files" ni tanlang
  4. "Clear data" tugmasini bosing
Keyin: Sahifani qayta yuklang va http://localhost:3000 ga kiring
